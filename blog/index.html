Estas viendo como se ve un blog

Este es el texto del blog. No tiene animaciones, ni efectos de carga, ni movimientos al pasar el ratón. Es una estructura sólida pegada a la izquierda para una lectura rápida.

Puedes añadir tantos párrafos como necesites. El ancho está limitado a 800px para que las líneas de texto no sean infinitas en pantallas panorámicas, pero el bloque siempre nace desde la izquierda.
